Store one or both keys. Each tool below picks its own model from a combined Claude + GPT list; the key used is the one for whichever provider that model belongs to. Test API key(s) Keys are stored unencrypted in your Zotero profile on this computer and are only ever sent to the matching provider over HTTPS. Article text is sent to the provider only when you explicitly run a command. Workflow: highlight everything in one color (e.g. blue) while reading, then run “Categorize Highlights with AI” — each highlight is recolored to its category color, the same taxonomy across all articles. Blue is reserved: it never appears in the taxonomy, so blue always means “not yet categorized”. Right-click an article → “Find further reading (AI)”. The model reads the paper and its bibliography, searches the web, and saves a child note of recommended sources. Defaults: Cmd/Ctrl + Shift + S to summarize, Cmd/Ctrl + Shift + H to categorize highlights, with one or more items selected in the library. Both actions are also available by right-clicking selected items. The two keys must be different.

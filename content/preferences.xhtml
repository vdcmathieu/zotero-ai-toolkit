Test API key Keys are stored unencrypted in your Zotero profile on this computer and are only ever sent to the selected provider over HTTPS. Article text is sent to the provider only when you explicitly run a summarization. Any model ID accepted by the provider's API can be entered. claude-opus-4-8 gives the highest-quality summaries; claude-sonnet-4-6 and claude-haiku-4-5 are faster and cheaper. Workflow: highlight everything in one color (e.g. blue) while reading, then run “Categorize Highlights with AI” — each highlight is recolored to its category color, the same taxonomy across all articles. Blue is reserved: it never appears in the taxonomy, so blue always means “not yet categorized”. Defaults: Cmd/Ctrl + Shift + S to summarize, Cmd/Ctrl + Shift + H to categorize highlights, with one or more items selected in the library. Both actions are also available by right-clicking selected items. The two keys must be different.
